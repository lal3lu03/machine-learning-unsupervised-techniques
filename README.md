# Machine Learning - Unsupervised Techniques

This repository contains coursework and exercise material for unsupervised machine learning techniques.

It includes a collection of exercise folders, notebooks, scripts, and supporting files used to explore core unsupervised learning methods such as clustering, dimensionality reduction, and exploratory data analysis.

---

## Repository overview

The repository is organized into multiple exercise units:

```text
.
├── UE1/
├── UE2/
├── UE3/
├── UE4/
├── UE5/
├── UE6/
├── UE7/
├── requirements.txt
├── run_all.py
└── README.md
```

Each `UE` folder contains material for one exercise unit.

---

## Topics covered

Depending on the exercise unit, the repository may include work related to:

* unsupervised learning
* clustering
* dimensionality reduction
* exploratory data analysis
* feature representations
* model comparison
* notebook-based experimentation
* Python-based machine learning workflows

---

## Setup

Install the required Python dependencies:

```bash
python3 -m pip install -r requirements.txt
```

---

## Running the exercises

Some exercises can be run individually from their corresponding `UE` folder.

The repository also contains a helper script:

```bash
python3 run_all.py
```

Use this only if the local environment and exercise dependencies are correctly installed.
